# Homework 4.4: GitHub Practice

This repository contains the completed Quarto document from Homework 4.3 that contained data visualizations showing airport traffic and an example of Monte Carlo integration. It provides insight into the effect that a prompt may have on ChatGPT's response and includes a reflection of my experience in Stat 184.

## Overview

The goal of this project is to provide students in Stat 184 with practice using GitHub. More specifically, the repository contains a Quarto document from Homework 4.3 with four insightful sections. The first section illustrates how the annual number of travelers at six major airlines changed from 2020-2025, while Section 2 explains how to implement Monte Carlo Integration with a relevant example. The project also explores how the quality of GenAI's response will change based on the level of detail in the prompt, and a brief reflection of my semester in Stat 184 can be found in Section 4.
